# GitHub Templates

GitHub-specific templates for issues and pull requests.

## Templates

| Template | Purpose |
|----------|---------|
| [ISSUE_TEMPLATE/](ISSUE_TEMPLATE/) | Issue templates (bug, feature, etc.) |
| [PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md) | PR template |

## Usage

Copy to your project's `.github/` directory:

```bash
cp -r templates/github/ISSUE_TEMPLATE /path/to/project/.github/
cp templates/github/PULL_REQUEST_TEMPLATE.md /path/to/project/.github/
```
