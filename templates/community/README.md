# Community Templates

Templates for open source project community files.

## Templates

| Template | Purpose |
|----------|---------|
| [CONTRIBUTING-TEMPLATE.md](CONTRIBUTING-TEMPLATE.md) | Contribution guidelines |
| [SECURITY-TEMPLATE.md](SECURITY-TEMPLATE.md) | Security policy |
| [SUPPORT-TEMPLATE.md](SUPPORT-TEMPLATE.md) | Support information |

## Usage

1. Copy the template to your project root
2. Remove the `-TEMPLATE` suffix
3. Customize placeholders for your project
