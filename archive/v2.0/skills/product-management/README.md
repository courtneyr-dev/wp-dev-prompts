# Product Management Skills

Skills for WordPress plugin and theme product management, adapted from [deanpeters/product-manager-prompts](https://github.com/deanpeters/product-manager-prompts).

## Skills

| Skill | Description |
|-------|-------------|
| [jobs-to-be-done.md](jobs-to-be-done.md) | Understand WordPress user needs through functional, social, and emotional jobs |
| [user-story-wordpress.md](user-story-wordpress.md) | Create user stories with WordPress personas and Gherkin acceptance criteria |
| [plugin-positioning.md](plugin-positioning.md) | Position WordPress plugins using Geoffrey Moore's framework |
| [plugin-persona.md](plugin-persona.md) | Create proto-personas for WordPress user segments |
| [problem-framing.md](problem-framing.md) | Frame problems from the WordPress user's perspective |
| [feature-hypothesis.md](feature-hypothesis.md) | Create testable hypotheses for WordPress plugin features |

## Usage

Reference these skills when:
- Planning a new WordPress plugin or theme
- Prioritizing feature requests
- Writing product requirements
- Understanding your target market
- Validating product decisions

### With Claude Code

```markdown
For product planning, apply:
- skills/product-management/jobs-to-be-done.md
- skills/product-management/plugin-positioning.md
```

## WordPress Personas Quick Reference

| Persona | Technical Level | Budget | Key Concern |
|---------|-----------------|--------|-------------|
| DIY Site Owner | Low | $0-100/yr | "Just works" |
| Content Creator | Medium | $50-200/yr | Efficiency |
| Developer | High | $500-2000/yr | Documentation, hooks |
| Agency Owner | Strategic | $5000+/yr | Scale, support costs |
| Enterprise | Strategic | $10,000+/yr | Compliance, integration |
