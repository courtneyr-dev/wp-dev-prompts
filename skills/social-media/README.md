### Key Principles

1. Hook in line 1 — no warm-up
2. Specificity builds credibility (names, numbers, tools)
3. End 70% of posts with natural engagement
4. Ask: "Would I bookmark this if someone else wrote it?"

## Content Mix

Target weekly distribution:
- 38% educational/how-to
- 29% personal updates
- 19% insights/observations
- 14% questions/engagement
