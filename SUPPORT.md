### Common Questions

**Q: Which platform should I use?**

See the [Platform Comparison](platforms/README.md) for recommendations based on your use case.

**Q: How do I use prompts with ChatGPT/Gemini?**

Web platforms don't have file access. Copy prompt content directly into your conversation, or see [platforms/chatgpt/](platforms/chatgpt/) and [platforms/gemini/](platforms/gemini/) for Custom GPT and system instruction setup.

## Reporting Issues

### Bug Reports

If you find a problem:

1. Search [existing issues](https://github.com/courtneyr-dev/wp-dev-prompts/issues) to see if it's already reported
2. If not, [open a new issue](https://github.com/courtneyr-dev/wp-dev-prompts/issues/new) with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Platform/tool you're using

### Feature Requests

Have an idea? [Open an issue](https://github.com/courtneyr-dev/wp-dev-prompts/issues/new) describing:
- What you'd like to see
- Why it would be useful
- Any implementation ideas

### Documentation Issues

Found a typo or unclear docs? Either:
- Open an issue describing the problem
- Submit a PR with the fix

## Response Time

This is a community project maintained in spare time. Please be patient:
- Issues are reviewed as time permits
- PRs may take a few days to review
- Complex changes may require discussion

## Security Issues

For security vulnerabilities, please do not open a public issue. Instead:
1. Check if the repository has a security policy
2. Report via GitHub's private vulnerability reporting if available
3. Or email the maintainer directly
